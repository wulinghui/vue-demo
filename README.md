# imoocmall

> Imooc Mall 
## 环境
mongodb
node
## 导入数据
建立db:dumall
集合: goods、users
导入数据: resource/dumall-goods.json、dumall-users.json

## Build server
node server/bin/www

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
